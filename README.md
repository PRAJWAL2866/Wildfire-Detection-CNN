# Wildfire Detection using CNN and Transfer Learning

## Project Overview

This project classifies images into:

- Fire
- Smoke
- Non-Fire

using Deep Learning techniques.

## Dataset

Forest Fire, Smoke and Non-Fire Image Dataset

Dataset Link:
https://www.kaggle.com/datasets/amerzishminha/forest-fire-smoke-and-non-fire-image-dataset

## Models Used

### Custom CNN
Accuracy: 88.30%

### ResNet50 Transfer Learning
Accuracy: 78.83%

| Model | Accuracy |

## How to Run

1. Install dependencies:
pip install -r requirements.txt

3. Download the dataset from Kaggle:
https://www.kaggle.com/datasets/amerzishminha/forest-fire-smoke-and-non-fire-image-dataset

3. Open wildfire-notebook.ipynb

4. Run all cells sequentially.

5. The notebook will:
   - Load the dataset
   - Preprocess images
   - Train the CNN model
   - Evaluate the model
   - Generate classification metrics
